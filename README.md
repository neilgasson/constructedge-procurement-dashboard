# ConstructEdge Procurement Intelligence — GitHub Pages

This folder is a dependency-free static export of the validated ConstructEdge
Procurement Intelligence website.

## Publish

1. Upload the contents of this folder to the root of the GitHub repository
   `constructedge-procurement-dashboard`.
2. In the repository settings, open **Pages**.
3. Select **Deploy from a branch**, choose the main branch and the `/ (root)`
   folder, then save.

The expected public address is:

`https://neilgasson.github.io/constructedge-procurement-dashboard/`

All navigation, data, downloads, styles, and scripts use relative paths. The
site does not require a build step, package manager, server-side runtime, or
external platform dependency.
